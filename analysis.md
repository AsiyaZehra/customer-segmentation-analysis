# Customer Segmentation Analysis

## Overview
Performed customer segmentation for an e-commerce brand (meltbeauty.net) 
using K-means clustering in Python to identify distinct customer groups 
and enable targeted marketing campaigns.

## Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- Google Analytics
- Excel

## What I Did
- Cleaned and prepared e-commerce transaction data
- Applied K-means clustering to segment customers by behavior and purchase patterns
- Identified 4 distinct customer segments (high-value, at-risk, new, dormant)
- Mapped segments to targeted marketing strategies

## Results
- Improved marketing engagement by **15%**
- Enabled more personalized campaign targeting across segments
